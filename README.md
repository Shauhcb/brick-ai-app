# 🧱 BRICK AI - Multi-Source Search Engine

BRICK AI is a powerful multi-source search engine that combines results from Google, Bing, Wikipedia, and AI-powered smart summaries.

## ✨ Features

- 🔍 **Multi-Source Search** - Get results from multiple search engines in one place
- 🌐 **Google Search** - Top results from Google
- 🔎 **Bing Search** - Results from Microsoft Bing (requires API key)
- 📚 **Wikipedia** - Article summaries and links
- 🤖 **AI Smart Search** - AI-generated comprehensive answers (requires Gemini API)
- 👤 **User Authentication** - Secure login and registration
- 🌙 **Dark/Light Theme** - Switch between themes
- 📜 **Search History** - Track your previous searches
- 💬 **Feedback System** - Send feedback to improve the app
